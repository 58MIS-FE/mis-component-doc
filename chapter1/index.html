Jquery 插件

Jquery 插件

  1. 引入assets/utils中Jquery
    <script src="../assets/js/jquery-1.12.4.js"></script>

  2. 引入assets/packages中对应的组件文件

    <script src="../assets/package/button.js"></script>

  3. Mackdown中书写对应文档及示例

书写示例可通过直接Mackdown当中书写HTML标签及<script></script>

Example

以下使用Button组件当中文档示例

Quick Start

通过示例应用组件

简单应用

$('#button_simple').button({
  text: '我是按钮'
})

指定type

$('#button_type').button({
  text: '我是特殊按钮',
  css: {
    background: 'blue'
  }
})
